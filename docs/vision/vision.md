## Application vision

Lenses Manager application

## Goal

Application is means to help with lenses, specifically so that app manages time frame of daily lenses, 

- how much user wears lenses
- application notification reminder to take lenses on and take them off


## User story

1) Select lenses type and wear timeframe

2) Record lenses wear, we select a timeslot that we plan to use the lenses, it's up to 16 hours
   
3) there should be also an option to enable notification, our app will notify user to wear lenses
- by the notification there should be an action button to actually "say ok" - lenses put on
- when time to wear lenses ends, there should be again a notification to remind user to take lenses off

here one important thing about algorithm to remind user to take off lenses, 
if user does not click on "lenses taked off", we should send notification again to take lenses off, and so on, so user does not forget and fall asleep with lenses on!!!

about notification to take lenses off, we should use some custom alert ringtone
